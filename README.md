## 📂 Cấu trúc dự án (MVVM)
- **Models**: Chứa các lớp thực thể dữ liệu (HocSinh, Lop,...)
- **ViewModels**: Xử lý logic nghiệp vụ và liên kết dữ liệu (Data Binding).
- **Views**: Giao diện XAML (UserControls cho từng biểu mẫu BM).
- **Services**: Các lớp kết nối Database SQL Server.
- **Resources**: Chứa Style Material Design và các icon của ứng dụng.
